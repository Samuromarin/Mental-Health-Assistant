# Autoestima Saludable

## Ejercicios diarios:
1. **Gratitud**: 3 cosas positivas del día
2. **Autocompasión**: Hablar como a un buen amigo
3. **Logros**: Reconocer pequeños éxitos
4. **Límites**: Decir "no" cuando es necesario

## Pensamientos útiles:
- "Soy suficiente como soy"
- "Merezco respeto y amor"
- "Los errores son oportunidades"
